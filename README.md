# Bitasmbl-Simple-Budget-Expenses-Dashboard-60e66a-b9bbb1a779c7

## Description
Build a web application that allows users to track their income and expenses, visualize monthly spending patterns with charts, and gain insights into their financial habits. Focus on simplicity, intuitive charts, and interactive summaries.

## Tech Stack
- SvelteKit
- Zustand

## Requirements
- Allow users to input income and expense entries
- Categorize expenses and show monthly totals
- Visualize spending patterns using interactive charts
- Allow users to edit or delete previous entries
- Provide a summary of monthly or yearly budgets

## Installation
bash

cd Bitasmbl-Simple-Budget-Expenses-Dashboard-60e66a-b9bbb1a779c7

## Usage
bash
npm run dev


## Implementation Steps
1. Initialize SvelteKit project and configure basic routes.
2. Set up Zustand store for income, expenses, and categories.
3. Implement forms to add, edit, and delete income and expense entries.
4. Create derived state for monthly totals and summaries.
5. Build interactive charts for spending patterns using stored data.
6. Implement monthly and yearly budget summary views.
7. Add basic UI layout focused on simplicity and clear summaries.
